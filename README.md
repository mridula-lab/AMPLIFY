# AMPLIFY: The Socratic Wizarding Quest
A Gamified AI-Learning Ecosystem for the AMD Slingshot 2026

> "AMPLIFY isn’t a tool that solves math; it’s an engine that builds mathematicians."

##  Project Overview
Most educational AI tools act as "Answer-Bots"—providing instant solutions that bypass the critical thinking phase required for learning. **AMPLIFY** restores the "Neural Struggle" by transforming mathematical problem-solving into an interactive RPG. 

Powered by **Google Gemini 2.5 Flash** and optimized for **AMD Ryzen™ AI**, AMPLIFY replaces math anxiety with gaming curiosity.

##  The Socratic Engine
Unlike traditional calculators, our AI acts as a **Socratic Mentor**. 
- **Context-Aware Hinting:** Instead of giving the answer, the Wizard analyzes your specific mistake (e.g., a forgotten negative sign) and provides a witty conceptual nudge.
- **Scaffolded Success:** Complex topics like Quadratic Equations are broken into three mini-quests: 
  1. The Discriminant (Calculation)
  2. Nature of Roots (Logic)
  3. The Final Mastery (Solution)

##  The Evolutionary Bestiary
To drive long-term engagement, academic progress is tied to a digital companion.
- Level 1:(0 xp) Ash Pebble - The Journey Begins
- Level 2:(150 xp) Flame Chick - Discriminant Calculation Mastery
- Level 3:(400 xp) Spark Drake - Nature of Roots Identification
- Level 4:(800 xp) Cloud-Breaker Wyvern - Complete Quadratic Solving
- Level 5:(1000 xp) Archmage’s Grand Mythos - Persistent Logical Excellence

##  Tech Stack & AMD Integration
- Engine: Python & Streamlit
- Intelligence: Google Gemini 2.5 Flash API
- Mathematical Rendering: LaTeX / MathJax
- Data Persistence: Persistent save-state via `wizard_save.txt`

###  AMD Edge Advantage
AMPLIFY is designed with the **AMD Ryzen™ AI** roadmap in mind. By utilizing the dedicated **NPU (Neural Processing Unit)** in Ryzen 7000/8000 series processors, we aim to:
1. Enable Local Inference: Run the Socratic Wizard offline and with zero latency.
2. Ensure Student Privacy: Keep sensitive educational data on the device, not the cloud.
3. Power Efficiency: Optimize high-frequency tutoring sessions for longer battery life on student laptops.
